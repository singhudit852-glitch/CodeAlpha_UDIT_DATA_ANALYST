# CodeAlpha_UDIT_DATA_ANALYST
This repo contains all the task that were assigned to me during my internship time in CodeAlpha 
This project demonstrates an end-to-end web scraping, data analysis, and sentiment analysis workflow using Python. It begins by scraping quotes and their authors from a public website (quotes.toscrape.com) using requests and BeautifulSoup. The extracted data is then cleaned, organized into a Pandas DataFrame, and saved as a CSV file. Subsequently, a comprehensive Exploratory Data Analysis (EDA) is performed to understand data distributions, such as quote lengths and author frequencies, visualized using Matplotlib and Seaborn. Finally, sentiment analysis is applied to each quote using TextBlob, categorizing them as positive, negative, or neutral, providing insights into the emotional tone of the collected content. This project showcases practical skills in data acquisition, manipulation, visualization, and basic natural language processing.
